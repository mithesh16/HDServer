
# HD-Server 

## Run Locally

Clone the project

```bash
  git clone https://github.com/mithesh16/HDServer
```

Go to the project directory

```bash
  cd HDServer
```

Install dependencies

```bash
  npm install
```
Create .env file as per sample

```bash
  vi .env
```

Run the backend server 

```bash
  npm run start
```
